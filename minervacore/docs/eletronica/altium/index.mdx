# Altium Designer

Bem-vindo à capacitação de Altium Designer. Selecione um dos tópicos abaixo para entrar no conteúdo do tema.

> Altium Designer é um software de design eletrônico utilizado para a criação de placas de circuito impresso (PCBs). 
> Ele oferece uma ampla gama de ferramentas para esquemáticos, layout de PCB, simulação e gerenciamento de projetos eletrônicos.
