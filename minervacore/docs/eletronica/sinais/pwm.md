# PWM

### [PWM](http://www.bosontreinamentos.com.br/eletronica/curso-de-eletronica/curso-de-eletronica-o-que-e-pwm-pulse-width-modulation/)

***PWM*** é uma sigla em inglês para ***Pulse Width Modulation***, ou em português, ***Modulação por Largura de Pulso***. O PWM funciona modulando o ciclo ativo (*duty cicle*) de uma onda quadrada. O conceito de funcionamento é simples. O controlador (fonte de tensão com PWM) entrega uma série de pulsos, gerados em intervalos de igual duração, que pode ser variada. Quanto mais largo o pulso, maior a quantidade de corrente fornecida à carga.

**Principais Características**:

- Permite o controle de potência por vias digitais;
- Simula um sinal analógico.

> Atente-se para capacidade da porta do seu controlador de fornecer um sinal PWM.
