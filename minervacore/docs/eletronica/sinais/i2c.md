# I2C

### [I2C](https://www.topgadget.com.br/howto/eletronica/visao-geral-e-beneficios-do-protocolo-e-barramento-i2c.htm)

![](https://i.imgur.com/uJNiOnp.png)

O sinal ***I2C*** utiliza apenas de dois pinos para se comunicar com todos os dispositivos. Os dois sinais usados são:
- Linha de dados serial (SDA)
- Relógio de dados serial (SCL)

Sendo a porta SDA recebendo o sinal SDA e a porta SCL recebendo o sinal SCL.

**Principais Características:**
- Realiza a comunicação de varios dispositivos através de apenas dois pinos, sendo possível devido a capacidade de cada um deles de interpretar o papel de sensor, atuador ou outro microcontrolador.
- Possui como desvantagem uma comunicação não tão rápida
- Cada dispositivo possui um endereço para sua identificação;
- Não é uma comunicação tão rápida;
