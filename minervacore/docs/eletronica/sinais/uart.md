# UART

### [UART](https://paginas.fe.up.pt/~hsm/docencia/comp/uart/)

![](https://i.imgur.com/NimX8Wc.png)

***UART*** (Universal Asynchronous Receiver/Transmitter), tem sua comunicação realizada de modo assíncrono (sem clock de sincronização), sendo realizada de forma que o TX de um vai no RX de outro; além de compartilharem do mesmo terra.

- Sendo:
A porta RX responsável por receber dados;
A porta TX responsável por enviar dados;
