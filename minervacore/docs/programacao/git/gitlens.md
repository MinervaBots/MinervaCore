# GitLens no VSCode

Basta ir na aba git no canto superior esquerdo dada pelo ícone ![InterfaceGit](https://imgur.com/B3Jq3WM.png)

## Como dar commit?

Existe duas maneiras de dar commit a primera é apertando direto o seguinte botão:

**Fig 4 - Como dar commit**

![Fig4](https://imgur.com/y2DTiht.png "Como dar commit")

Se fizer isso receberá um alerta que não tem nada selecionado para commit ou "Staged" e ai você pode simplesmente dar um "Yes" no alerta que ele vai commitar todas as mudanças.

A outra maneira é selecionar primeiro oque quer que vá para o commit, basta ir no arquivo e clicar no +. Para desfazer aquela mudança no seu repositório clique na setinha de voltar, ambos representados na imagem abaixo:

**Fig 5 - Stage & Discart**

![Fig5](https://imgur.com/DEKWFpT.png "Stage & Discart")

## Não se esqueça de dar Push!!

Depois que der o commit você ainda precisa dar Push para as suas mudanças irem para o repositório remoto basta clicar nos "..." que você poderá dar Push ou até mesmo outras funções úteis como Pull ( Que atualiza o seu repositório local com o que está no repositório remoto) dentre outras, confira na imagem abaixo:

**Fig 6 - Como dar Push**

![Fig6](https://imgur.com/7YxrYKy.png "Como dar Push")
