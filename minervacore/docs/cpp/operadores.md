# Operadores

## Operadores Aritméticos

``` c++ 

+ 	adição 	        x + y 	
- 	subtração     	x - y 	
* 	multiplicação 	x * y 	
/ 	divisão 	    x / y 	
% 	módulo 	        x % y 	
	
```

## Operadores Lógicos

``` c++ 

&&  	retorna verdadeiro se as duas 
        sentenças são verdadeiras ou falsas	                
        (x < 5 &&  x < 10 )	
        
|| 	    retorna verdadeiro se uma 
        sentença é verdadeiras
        (x < 5 || x < 4 	)
        
! 	    converte o resultado
        !(x < 5 && x < 10)


```

## Operadores Comparadores

``` c++ 
== 	igual 	                        x == y 	
!= 	diferente	                    x != y 	
> 	maior que     	                x > y 	
< 	menor que 	                    x < y 	
>= 	maior que ou igual    	        x >= y 	
<= 	menor que ou igual  	        x <= y

```
