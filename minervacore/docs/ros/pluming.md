# Plumbing

É a forma com que as ferramentas do ROS se organizam, como já dito, a utilização do ROS se deve por uma rede onde há uma troca de informação dos elementos que compõem essa rede. Não se atentem ao conteúdo da imagem abaixo, é apenas uma ilustração para mostrar o plumbing: como cada elemento é identificado, através desses quadrados e círculos; e na troca de informações entre esses elementos através das setas.
